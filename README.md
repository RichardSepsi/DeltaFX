# DeltaFX website source code

DeltaFX is a website that I made to provide digital 3d assets (PBR Materials, Models, etc.) under the cc0 (public domain) license.
This repo contains the source code for the website.

## How to run

The website is built using plain HTML/CSS/JS/PHP.
It can be ran locally, but for downloading to work, it needs to be hosted on a server as it's handled using php.
During development I used this php server vscode extension: https://marketplace.visualstudio.com/items?itemName=brapifra.phpserver
